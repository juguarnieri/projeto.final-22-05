ECO RACING

Projeto carrinho arduíno

O projeto final consiste na idealização e construção de um um carrinho motorizado que seja capaz de navegar por um circuito temático ambiental. A partir dele, será possível cumprir com o desafio proposto de conscientização eco sustentável através da automação e programação.

Detalhes do projeto

Sustentabilidade

Um dos objetivos do projeto é a sustentabilidade e preocupação com o meio ambiente. Com isto, nosso carrinho conta com partes feitas de materiais recicláveis.

Aerodinâmica

Outro objetivo deste projeto é a aerodinâmica, para que o trajeto consiga ser concluído em menor tempo, além de seu design.

Programação

E por último, o desenvolvimento de um código para o arduino, fazendo com que o carro se movimente além de algumas manobras para facilitar o percurso.
